# alexhermosillo.com.mx – Personal Website

## Overview

This repository contains the source code for my personal website and blog:
**alexhermosillo.com.mx**

The purpose of this project is to document my development process, share my projects, write about my work experiences, travels, and technical topics, and continuously improve the website over time.

This project is intentionally built using **pure HTML, CSS, and JavaScript** to fully understand web fundamentals before moving to frameworks or more complex architectures.

---

## Deployment Workflow

This project uses an **automatic deployment workflow** connected to my personal hosting server.

**Workflow process:**

1. Changes are developed and tested locally.
2. Changes are committed and pushed to the `main` branch.
3. The server automatically pulls the latest version.
4. The website updates automatically.

This allows me to work from anywhere and keep the website always updated with the latest stable version.

## Website Architecture

The website follows a **basic modular architecture** using HTML for structure, CSS for styling, and JavaScript for interactivity.

### Architecture Philosophy

The HTML files are used only for structure and content.
All styling is separated into CSS files, and JavaScript is separated for logic and dynamic behavior.

This keeps the project:

* Organized
* Maintainable
* Scalable
* Easy to debug
* Easy to redesign without changing content

### Basic Structure Example

```
/root
    index.html
    /css
        main.css
        home.css
        blog.css
        animations.css
    /js
        main.js
        animations.js
        menu.js
    /images
    /blog
        post1.html
        post2.html
```

### HTML

Used only for:

* Layout
* Content
* Semantic structure
* Linking CSS files
* Linking JavaScript files

### CSS

Separated by responsibility:

* Main styles
* Page styles
* Animations
* Layout
* Responsive design

### JavaScript

Used for:

* Animations
* DOM manipulation
* Navigation menu
* Dynamic content
* Future backend/API integration
